1.Тема «Применение и разработка машинного обучения для автоматизированного анализа научных публикация в области динамики потоков».
2. Направление/аспект применения ИИ: Использование методов машинного обучения и методов обработки естественного языка (текстовый формат) для анализа и извлечения ключевых элементов из огромного числа научных текстов.

3. Аргументация выбора: Учусь в исследовательском институте, и проведение литературного обзора по необходимой тематике занимает большое количество времени. Подобный ручной анализ является неэффективным в наше время, когда прогресс буквально "летит". Также есть возможность не успеть опубликовать результаты первым по сравнению с другими командами ученых. ИИ позволит быть на шаг впереди в научной гонке.
4. Ожидаемый результат: Сокращение времени на обработку и анализ данных научных публикаций. Возможность выявления закономерностей между различными методами и создание интерактивных карт для наглядного показа структуры направления тематики, связанной с динамикой потоков.
